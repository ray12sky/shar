# 💊 Virtual Lab: Medication Tracker

## 📌 Overview
This project is a Streamlit-based web application designed to simulate a virtual lab for tracking medications.

## 🚀 Features
- Add and manage medications
- Track dosage and timing
- Drug interaction checker
- Reminder simulation

## 🛠️ Tech Stack
- Python
- Streamlit
- Pandas

## ▶️ How to Run
1. Clone the repo
2. Install dependencies:
   pip install -r requirements.txt
3. Run:
   streamlit run app.py

## 🎯 Future Scope
- AI-based interaction prediction
- Integration with wearable devices
- Real-time notifications
